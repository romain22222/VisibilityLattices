# AdaFit models

Official source: `https://github.com/Runsong123/AdaFit`.

Place trained AdaFit checkpoint(s) in this directory or point `visibilityLattices` to another location.

The bridge can also use checkpoints directly from an AdaFit checkout, e.g.:

- `trained_model/AdaFit/my_experiment_model_599.pth`
- `trained_model/AdaFit/my_experiment_params.pth`

If checkpoint loading or inference fails, the application reports an error directly (no fallback mode).
